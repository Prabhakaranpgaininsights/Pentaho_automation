🏠 Overview
Pentaho AI is an intelligent ETL file generation platform powered by Anthropic's Claude AI. It automates creation of Pentaho Data Integration files — KTR transformations, KJB jobs, and SQL — from natural-language descriptions or uploaded schemas.
⚙️
File Generator
AI-powered KTR + KJB generation from step instructions.
🔄
KTR / KJB Upload
Upload instructions → download ready-to-use Pentaho files.
🗄
Table Builder
AI-driven schema creation with smart SQL type inference.
🔗
Join Builder
Multi-table JOIN pipeline builder with live SQL execution.
📊
Dashboard
Live metrics: API calls, tokens, success rate, history.
📦
Downloads
Manage and bulk-download all generated files as ZIP.
💡
Tech stack: Django 4.x + Python 3.10 + PostgreSQL + Anthropic Claude API
📥 Quick Start Summary
1
Install Python 3.10+ and create a virtual environment
See the Python Install section for platform-specific instructions.
2
Install Django and dependencies via pip
pip install -r requirements.txt
3
Configure settings.py with your API key and DB credentials
Set ANTHROPIC_API_KEY, ANTHROPIC_MODEL, and DATABASES.
4
Run migrations and create a superuser
python manage.py migrate && python manage.py createsuperuser
5
Start the development server
python manage.py runserver → http://127.0.0.1:8000/

📁 Project Setup
⚙️ settings.py — Required Config
# Anthropic AI ANTHROPIC_API_KEY = "sk-ant-your-key-here" ANTHROPIC_MODEL = "claude-sonnet-4-20250514" # PostgreSQL DATABASES = { "default": { "ENGINE": "django.db.backends.postgresql", "NAME": "pentaho_db", "USER": "postgres", "PASSWORD": "your-password", "HOST": "localhost", "PORT": "5432", } } MEDIA_URL = "/media/" MEDIA_ROOT = BASE_DIR / "media"
⚠️
Never commit ANTHROPIC_API_KEY to version control. Use .env files.
🗄 Database Setup
🐘 PostgreSQL — Create DB
CREATE DATABASE pentaho_db; CREATE USER pentaho_user WITH PASSWORD 'yourpassword'; GRANT ALL PRIVILEGES ON DATABASE pentaho_db TO pentaho_user;
🔄 Run Django Migrations
python manage.py makemigrations python manage.py migrate
👤 Create Superuser
python manage.py createsuperuser

🔧 Troubleshooting
Common Issues
⚠️
API key invalid: Ensure your key starts with sk-ant- and has sufficient credits.
⚠️
DB connection error: Verify PostgreSQL is running and credentials in settings.py are correct.
ℹ️
Timeout errors: Increase the timeout value in API Settings modal for large generations.
🔒 Security Notes
⚠️
Never expose your Anthropic API key in client-side code or version control.
⚠️
SSL verification is disabled by default for local development. Enable for production.
✅
Use Django's built-in CSRF protection for all POST endpoints.
❓ FAQ
Can I use both Pentaho and Talend at the same time?
Yes — use the platform switcher in the top bar to toggle between platforms. Each platform has its own stats URL and file types.
Where are generated files stored?
Files are stored in the MEDIA_ROOT directory defined in settings.py. Access them via the Downloads modal.
Which Claude model should I use?
Claude Sonnet 4 is recommended for best speed/quality balance. Opus 4 for complex pipelines, Haiku 4.5 for fast previews.
